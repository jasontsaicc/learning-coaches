# Language

## Default Language

English. Problems, code, and complexity discussion run in English so sessions double
as reading practice for the interview.

## Ramp Policy

Light anchor policy, no scheduled ramp: each pattern's key insight (the aha moment)
may use Traditional Chinese anchor terms, matching the student's 30/70 study style.
Everything else stays English. No transition schedule; Chinese is allowed exactly at
the insight moment and in the student's own notes.
